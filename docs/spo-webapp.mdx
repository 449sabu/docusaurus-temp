# SPO-WebApp
- このドキュメントは、誰でも簡単にSPO-WebAppを利用できるように作成しています。
「わかりにくい」や「難しい」という改善提案はピンポイントでご指摘いただけると幸いです。  
- 免責事項
非エンジニアが独学でコードを書いています。サイトの安全性を重視し言語はTypeScriptを使用しています。

## 事前準備
- [Googleアカウント](https://www.google.com/intl/ja/account/about/)の作成（新規作成を推奨）  
プールへのお問い合わせ先にGmailを使用します。メールアドレスを一般公開するとスパムメールなどが送られてくる場合がありますので、普段使いのメールアドレスではなく専用のメールアドレスを新規に作成することを推奨します。
- [Githubアカウント](https://github.com/)の作成
