# SPO-WebApp
- このドキュメントは、誰でも簡単にSPO-WebAppを利用できるように作成しています。
「わかりにくい」や「難しい」という改善提案はピンポイントでご指摘いただけると幸いです。  
- 免責事項
  - 非エンジニアが独学でコードを書いています。
  - サイトの安全性を重視し言語はTypeScriptを使用しています。

## 事前準備
- [Googleアカウント](https://www.google.com/intl/ja/account/about/)の作成（新規作成を推奨）  
プールへのお問い合わせ先にGmailを使用します。メールアドレスを一般公開するとスパムメールなどが送られてくる場合がありますので、普段使いのメールアドレスではなく専用のメールアドレスを新規に作成することを推奨します。
- [Githubアカウント](https://github.com/)の作成
プール運営で使用しているアカウントでかまいません。リポジトリをGithubに複製し、[Versel](https://vercel.com/login?next=)でサイトを公開するときに使用します。

## 利用方法

### GoogleアプリKeyを発行する（編集中です）
これから作成するサイトには、メールアドレスを利用したお問い合わせフォームが標準搭載されています。  
Googleアカウント作成手順はこのマニュアルでは省略いたします。Googleアカウントをお持ちでない場合は、新規作成してから手順を進めてください。  

### リポジトリを複製する
はじめに、自分のGithubに「SPO-WebApp」のコードを複製します。
Githubにログインした状態で https://github.com/449sabu/spo-webapp に移動し、右上の「Fork」をクリック。  

![img](/img/fork.png)

「Create Fork」をクリックします。  

![img](/img/create-fork.png)

### サイトを公開する
サイトを公開するためにVercelというサーバーを提供しているサービスを利用します。  
VercelへはGithubアカウントでログインできます。  

![img](/img/vercel.png)

電話認証に進むので、居住国を選択し電話番号を入力し送られてくるパスコードを入力してください。  

![img](/img/vercel2.png)

電話認証を終了すると、「どのリポジトリを読み込むか」というポップアップが出てくるので、「All repositories」が選択された状態で「Install」をクリック。  

![img](/img/vercel3.png)

Vercelにログインできると以下のような画面になるので、「Continue with GitHub」をクリック。

![img](/img/vercel4.png)

リポジトリの一覧から「spo-webapp」を探し、「Import」をクリック。

![img](/img/vercel5.png)

「Environment Variables」を選択し、環境変数を設定します。  
以下の表を参考にNameとValueにそれぞれ入力し「Add」をクリック。  

|  Name  |  Value (Will Be Encrypted)  |  必須  |
| ---- | ---- | ---- |
|  NEXT_PUBLIC_POOL_ID  |  自分のPool ID（pool...で始まるもの）  |  ◎  |
|  MAIL_TO  |  自分のGmailアドレス  |  ◎  |
|  MAIL_PASS  |  Googleアプリkey  |  ◎  |

![img](/img/vercel6.png)

必要な環境変数を入力出来たら「Deploy」をクリック。


## リポジトリを最新に保つ
「SPO-WebApp」はバグ修正や、新機能の追加など継続的に開発をしていきます。
Twitterでのアップデート通知を行いますので、以下の手順で更新を行ってください。

Githubにログインし、自分のリポジトリを開く。  
「Sync fork」を選択し、「Update branch」をクリック。  

![img](/img/update-code.png)

「Successfully fetches ...」というメッセージが出たら更新完了です。  
Verselがリポジトリの更新を自動で認識し、ウェブサイトが更新されます。  

![img](/img/update-code2.png)
